[B][SIZE=4]SneakPeak - Precision Waveform Item Editor for REAPER[/SIZE][/B]

[I] ⚠️ Current release: macOS Apple Silicon only. Windows/Linux builds coming soon.[/I]

[IMG]https://raw.githubusercontent.com/b451c/SneakPeak/main/docs/images/sneakpeak-hero.png[/IMG]

A native C++ extension that gives you a detailed, dockable waveform editor for any media item. Click an item in REAPER's arrange view, and SneakPeak instantly shows you a full-featured waveform editor.

[LIST]
[*]Copy toolbar icons to your REAPER toolbar icon folder

[B][SIZE=3]Three Editing Modes[/SIZE][/B]

[LIST]
[*][B]REAPER mode[/B] - Click an item, see its waveform. Non-destructive editing through REAPER API with full undo.
[*][B]SET mode[/B] [SIZE=2][I](v1.7)[/I][/SIZE] - Select multiple items, press T to lock them as a working set. Non-destructive editing as one continuous waveform with ripple editing, selection-aware gain, and grouping.
[*][B]Standalone mode[/B] - Drag & drop audio files for offline destructive editing. Up to 8 tabs with independent undo stacks.
[/LIST]

[B][SIZE=3]Features[/SIZE][/B]

[I]Waveform & Display[/I]
[LIST]
[*][B]Precision waveform[/B] - Peak + RMS display, dB scale, zero-crossing line
[*][B]Clipping visualization[/B] [SIZE=2][I](v1.7)[/I][/SIZE] - Peaks above 0dB draw in red proportional to severity
[*][B]Deep zoom[/B] - Horizontal + vertical zoom with minimap overview
[*][B]Spectral analysis[/B] - Real-time FFT spectrogram with frequency band selection
[*][B]Multi-item view[/B] - Mix, Layered per Item, Layered per Track modes with gap visualization and cross-segment editing [SIZE=2][I](v1.8)[/I][/SIZE]
[*][B]Ruler time format[/B] [SIZE=2][I](v1.7)[/I][/SIZE] - Toggle relative vs absolute REAPER timeline time
[/LIST]

[I]Editing & Processing[/I]
[LIST]
[*][B]Audio editing[/B] - Cut, copy, paste, normalize, reverse, gain, DC remove, silence
[*][B]Fade handles[/B] - Continuous curvature control matching REAPER's native fades. Real-time bidirectional sync [SIZE=2][I](v1.8)[/I][/SIZE]
[*][B]Selection-aware gain[/B] [SIZE=2][I](v1.7, improved v1.8)[/I][/SIZE] - Gain knob respects selection across all view modes with live preview
[*][B]Timeline View[/B] [SIZE=2][I](v1.8)[/I][/SIZE] - After cutting an item, all fragments stay visible with gaps preserved. Continue editing with zoom preserved.
[*][B]Option+click segment snap[/B] [SIZE=2][I](v1.8)[/I][/SIZE] - Instantly select a segment for gain re-adjustment without new splits
[*][B]LUFS normalization[/B] - -14 / -16 LUFS targets
[*][B]Snap to zero-crossing[/B] - Click-free edits
[*][B]Markers[/B] - Add, edit, delete, navigate markers and regions
[/LIST]

[I]Working Set (v1.7)[/I]
[LIST]
[*][B]Persistent editing set[/B] - Lock items with T, click away and come back - auto-restores
[*][B]Ripple edit[/B] - Delete pulls items together to close gaps
[*][B]Group Set Items[/B] - Group edited items for easy timeline manipulation
[*][B]Bidirectional cursor sync[/B] - Click in SneakPeak scrolls REAPER, click on timeline updates SneakPeak
[/LIST]

[I]File Management & Integration[/I]
[LIST]
[*][B]Standalone editing[/B] - Drag & drop any audio file, up to 8 tabs, independent undo
[*][B]Smart Save[/B] - Ctrl+S with overwrite confirmation, auto _edit.wav for MP3/FLAC, Save As
[*][B]Drag export[/B] - Drag selection outside window to place on REAPER timeline. Alt+drag for Finder export [SIZE=2][I](v1.8)[/I][/SIZE]
[*][B]Floating or docked[/B] [SIZE=2][I](v1.8)[/I][/SIZE] - Starts floating (resizable), dock/undock via right-click menu. Position remembered.
[*][B]Metering[/B] - Peak (PPM), RMS (AES/EBU), VU modes with master output option
[*][B]Zero dependencies[/B] - Pure C++ (REAPER SDK + WDL/SWELL)
[/LIST]

[B][SIZE=3]Keyboard Shortcuts[/SIZE][/B]

[CODE]
Space         Play / Pause
T             Toggle Working Set mode (v1.7)
S             Split at cursor (REAPER/SET mode)
G             Toggle gain panel
M             Add marker
Shift+M       Add region
Tab           Next marker
Escape        Exit Working Set / clear selection

Ctrl+Z        Undo
Ctrl+X/C/V    Cut / Copy / Paste
Ctrl+A        Select All
Ctrl+N        Normalize
Ctrl+S        Save (standalone)
Ctrl+Shift+S  Save As (standalone)
Delete / E    Delete selection

Scroll        Horizontal zoom
Shift+Scroll  Vertical zoom
Ctrl+Scroll   Pan left/right
Home / End    Go to start / end
[/CODE]

[B][SIZE=3]Installation[/SIZE][/B]

[B]ReaPack (recommended):[/B]
1. In REAPER: [B]Extensions > ReaPack > Import repositories...[/B]
2. Paste: [CODE]https://raw.githubusercontent.com/b451c/SneakPeak/main/index.xml[/CODE]
3. [B]Extensions > ReaPack > Browse packages[/B], search "SneakPeak"
4. Right-click > Install, restart REAPER

[B]Manual install:[/B] Download from [URL="https://github.com/b451c/SneakPeak/releases/latest"]GitHub Releases[/URL], copy to UserPlugins folder, restart REAPER.
